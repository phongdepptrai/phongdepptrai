### SAT Research: SALB-3PM

SALB-3PM (Simple Assembly Line Balancing with Power Peak Minimization) targets task assignment under cycle-time and precedence constraints while minimizing peak simultaneous power usage.

| Project | Focus |
|---|---|
| [SAML3P](https://github.com/phongdepptrai/SAML3P) | Foundational SAT solver suite with staircase CNF encoding |
| [P3SAML3P](https://github.com/phongdepptrai/P3SAML3P) | Multi-strategy platform with parameters `r_max` and `R_max` plus comparative solver pipelines |

P3SAML3P strategies in `run_launcher.py`:

| Strategy | Description |
|---|---|
| Base | Plain staircase feasibility |
| Staircase-1-3 | CardNet-based cardinality encoding |
| AtmostK | At-most-k PB constraint |
| Incremental | Ladder-variable peak tightening using PBEnc/binmerge |
| Inheritant | Warm-start upper bound from previous `r_max - 1` or `R_max - 1` runs |
| MaxSAT | RC2/WCNF via PySAT |
